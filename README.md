# ABadyal

This repository contains selected Computer Science material for the
`ABadyal` sTeX/MMT archive.

- Dynamic viewer base: `https://mmt.abadyal.com/:sTeX/browser?archive=ABadyal&filepath=index.xhtml`
- Public repo: `https://github.com/ArjunBadyal/ABadyal`

The repository is prepared for an MMT server and does not publish static XHTML
or PDF exports.
